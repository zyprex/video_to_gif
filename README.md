# Video to GIF

Features:

1. Extrat a frame of video to image.
2. Record part of video to another video (webm).
3. Convert video to gif.
4. Convert images to gif.

Incompatibilities:

- Test dose NOT passed on mobile firefox browser.
- Some very old browser.
- Low-end device.
